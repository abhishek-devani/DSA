# 01 Merge Sort

## Problem Description

```markdown
Given an integer array A, sort the array using Merge Sort.
```

---
## Problem Constraints

```
1 <= |A| <= 10^5
1 <= A[i] <= 10^9
```

---
## Input Format

```
First argument is an integer array A.
```

---
## Output Format

```
Return the sorted array.
```

---
## Example Input

```markdown
Input 1:
> A = [1, 4, 10, 2, 1, 5]

Input 2:
> A = [3, 7, 1]
```

---
## Example Output

```markdown
Output 1:
> [1, 1, 2, 4, 5, 10]

Output 2:
> [1, 3, 7]
```

---
## Example Explanation

### Explanation 1 & 2:

```markdown
Return the sorted array.
```
